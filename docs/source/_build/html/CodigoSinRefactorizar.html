Código No Refactorizado¶

Versión sin refactorizar del simulador de dados.

Esta versión cuenta con bad smells evidentes como: - Código duplicado en exceso. - Uso de variables globales y estado compartido. - Falta de separación de responsabilidades. - Simulaciones ineficientes que recalculan información constantemente.
