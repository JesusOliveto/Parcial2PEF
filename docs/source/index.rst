Proyecto 7 - Simulación de un Juego de Dados
=============================================

Bienvenido a la documentación del Proyecto 7 - Simulación de un Juego de Dados.

Descripción del Proyecto
------------------------
Este proyecto simula un juego de dados con diferentes modalidades y reglas, permitiendo 
análisis estadísticos de los resultados.

Características principales:
- Simulación de lanzamientos de dados
- Múltiples modalidades de juego
- Análisis estadístico de resultados
- Interfaz con Streamlit

Inicio Rápido
-------------

Para comenzar a usar el proyecto:

.. code-block:: bash

   # Instalar dependencias
   pip install -r requirements.txt

   # Ejecutar la aplicación Streamlit
   streamlit run implementacionRefactorizadoStreamlit.py
